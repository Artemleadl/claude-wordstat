# Wordstat API — справочник методов

Базовый адрес: `https://api.wordstat.yandex.net`
Транспорт: HTTPS, формат — JSON.
Авторизация: заголовок `Authorization: Bearer <OAuth-токен>`.
Все методы вызываются как `POST /v1/<method>` с JSON-телом.

Источник: https://yandex.com/support2/wordstat/ru/content/api-structure

> Примечание: точные имена некоторых необязательных полей (`period`, состав
> массива похожих запросов) Яндекс может корректировать. Клиент `wordstat.py`
> написан устойчиво к вариациям (`associations` / `searchedWith` / `similar`).
> При первом живом запросе сверь фактический JSON-ответ с разбором в скрипте.

## /v1/topRequests

Популярные запросы, содержащие фразу, и похожие запросы (за последние 30 дней).

Запрос:
```json
{
  "phrase": "купить кроссовки",
  "regions": [213, 2],
  "devices": ["phone"],
  "numPhrases": 200
}
```
- `phrase` (обяз.) — поисковая фраза, поддерживает операторы `" ! [] + -`.
- `regions` (необяз.) — массив ID регионов.
- `devices` (необяз.) — `all` | `desktop` | `phone` | `tablet`.
- `numPhrases` (необяз.) — число фраз в ответе, по умолчанию 50, максимум 2000.

Ответ:
```json
{
  "requestPhrase": "купить кроссовки",
  "totalCount": 123456,
  "topRequests": [
    {"phrase": "купить кроссовки мужские", "count": 45000},
    {"phrase": "купить кроссовки nike", "count": 30000}
  ],
  "associations": [
    {"phrase": "купить кеды", "count": 12000}
  ]
}
```
- `totalCount` — общее число запросов, содержащих все слова фразы в любом порядке.
- `topRequests[]` — пары `phrase` / `count`.
- похожие/ассоциации могут приходить отдельным массивом.

## /v1/dynamics

Динамика числа запросов по фразе во времени.

Запрос:
```json
{"phrase": "купить елку", "period": "monthly", "regions": [225]}
```
- `period` — детализация: `daily` | `weekly` | `monthly`.

Ответ:
```json
{
  "dynamics": [
    {"date": "2025-12", "count": 98000, "share": 0.012}
  ]
}
```
- `date` — период, `count` — число запросов, `share` — доля от всех запросов к Яндексу.

## /v1/regions

Распределение числа запросов по регионам мира (за последние 30 дней).

Запрос:
```json
{"phrase": "доставка пиццы"}
```

Ответ:
```json
{
  "regions": [
    {"regionId": 213, "count": 50000, "share": 0.34, "affinityIndex": 145}
  ]
}
```
- `affinityIndex` — индекс интереса: >100 означает повышенный интерес региона
  к фразе относительно среднего.

## /v1/getRegionsTree

Дерево/список поддерживаемых регионов с их ID и названиями. Тело запроса пустое `{}`.

## /v1/userInfo

Информация о квоте: лимиты и остаток. Тело запроса пустое `{}`.

## Лимиты

- Личная квота (по умолчанию): 10 запросов/сек, 1000 запросов/сутки.
- `429 Quota limit exceeded, try again later. Time to refill: N seconds` — личная квота.
- `503 Service unavailable, try again later` — общая квота API.
